# Power BI – Data Professional Survey Dashboard

## Overview
This project presents an interactive Power BI dashboard analyzing global survey data from data professionals. The objective was to extract meaningful insights related to salary distribution, job roles, geographic trends, and work satisfaction using business intelligence techniques.

📄 **Dashboard Preview:**
[View PDF Version](data-professional-survey-dashboard.pdf)

---

## Project Objectives
- Analyze salary trends across different job roles
- Compare survey responses by country
- Identify satisfaction levels with salary and work-life balance
- Design interactive KPIs for business insights

---

## Dashboard Features
- Country-wise distribution of survey participants
- Average salary breakdown by job title
- Programming language preference analysis
- Difficulty level to break into data roles
- KPI cards (Total Survey Takers, Average Age)
- Interactive filters and slicers

---

## Data Processing
- Data cleaning and transformation using Power Query
- Data modeling and relationship management
- KPI calculation and metric creation
- Visual storytelling using charts and gauges
